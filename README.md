1. 直角三角形：其中有兩個邊的平方和等於第三邊的平方。

2. 鈍角三角形：其中有兩個邊的平方和小於第三邊的平方。

3. 銳角三角形：任兩邊的平方和大於第三邊的平方。

輸入說明：
輸入三個整數。

輸出說明：
顯示直角三角形(Right Triangle)、鈍角三角形(Obtuse Triangle)、銳角三角形(Acute Triangle)或無法構成三角形(Not Triangle)。
